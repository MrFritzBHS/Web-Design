My Simple Site

This site has a simple, professional design to practice our CSS skills.

We are currently learning how to use **tag selectors** like p, em, and strong, and **class selectors** like .slogan and .highlight.

Learning Goals

  1. Match the final appearance of the website.
  2. Correctly use **tag selectors**.
  3. Correctly use **class selectors**.

CSS Properties Review

Property Description
font-size The size of the text
color The color of the text
background-color The color behind the element
